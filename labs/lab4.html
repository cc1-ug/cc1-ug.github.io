Ejercicio #1: (Nombre del archivo: ejercicio1.py)
Para este ejercicio se le pide que dado un numero N y un numero M leidos del usuario, cree una lista de length N, que este llena de M, y luego la despliegue. Para este ejercicio debe usar ciclos (puede usar while o for).
Ejemplo: 

			Ingrese N: 5
			Ingrese M: 3
			Lista:
			[3, 3, 3, 3, 3]

Ejercicio #2: (Nombre del archivo: ejercicio2.py)
Para este ejercicio se le pide que haga un programa en Python, que lea del usuario un String, y luego lea un caracter (Asegurese que el segundo elemento ingresado sea un String de un solo caracter). Despues cree una lista con los indices o posiciones en el String en las que aparezca el caracter, la despliegue y luego, encontrando el length de esa lista, despliegue cuantas existencias del caracter se encontraron en el String. Ejemplo: 

			Ingrese un String: palabra
			Ingrese un Caracter: a
			Posiciones de el caracter en el String:
			[1, 3, 6]
			Exitencias: 3
Nota importante: Para este ejercicio no puede usar while, solo for y DEBE construir la lista que se le pide.

Ejercicio #3: (Nombre del archivo: ejercicio3.py)
Para este ejercicio se le pide que implemente lo siguiente:

    (1) Lea un numero N de usuario.
    (2) Si el numero es entero y mayor que 0 : Genere N numeros aleatorios entre 1 y 100, y metalos en una lista, ordenados ascendentente.
    (3) Conforme va generando un numero aleatorio nuevo, debe desplegar el numero generado y desplegar la lista con el numero ya ordenado, todo esto debe hacerse en orden.
Para este ejercicio puede usar whiles y fors como le convenga. Recuerde que puede definir ciclos, dentro de ciclos.
Ejemplo: 
		Ingrese un numero N: 5
		Numero #1 generado: 11
		[11]
		Numero #2 generado: 34
		[11, 34]
		Numero #3 generado: 6
		[6, 11, 34]
		Numero #4 generado: 72
		[6, 11, 34, 72]
		Numero #5 generado: 15
		[6, 11, 15, 35, 72]
